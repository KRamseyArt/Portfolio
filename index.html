COOK•ALONG APP

HTML | CSS | JavaScript | jQuery | TheMealDB API | YouTube API

This recipe finding app eliminates lengthy historical accounts of a recipe's origins, and presents multiple options on how to make a specific dish. Users can find meals by looking for key ingredients, regional origins, and key terms. They are then shown a list of ingredients most commonly used in this dish's preparation, and several instructional videos that may vary from chef to chef.
